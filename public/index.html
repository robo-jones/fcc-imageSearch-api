Image Search Abstraction API

User Story: I can get the image URLs, alt text and page urls for a set of images relating to a given search string.

User Story: I can paginate through the responses by adding a ?offset=2 parameter to the URL.

User Story: I can get a list of the most recently submitted search strings.

Image Search API:

Endpoint:
https://splendid-bit.glitch.me/api/search
Parameters:
  • q: search query (required)
  • offset: first result to return (1-indexed)
  • num: number of results to return (1-10)

Example usage:

https://splendid-bit.glitch.me/api/search?q=kittens&offset=5&num=2

Recent Searches API:

Endpoint:
https://splendid-bit.glitch.me/api/search/recent
Parameters:
  • start: first result to return (0-indexed)
  • num: number of results to return (1+)

Example usage:

https://splendid-bit.glitch.me/api/search/recent?num=3&start=1

Example output:
